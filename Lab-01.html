The Data Set

We will continute to use the Overseas merchandise trade dataset which was introduced in Lab Zero.

For this lab we will have use a subset of the “Imports” dataset which has all imports of goods with the Harmonised Systems (HS) code starting with 8703 which denotes various types of vehicles. You can find the files on the VMs in the directory /course/data/trade/vehicles. Each file corresponds to one year (files are named vehicles-YYYY.csv) from 2000 to 2021.

For this lab, you will need to download or copy the files to the same directory as your R Markdown file. If you are on the FlexIT desktop, you can download and extract all files into the current directory using this command:

   curl https://www.stat.auckland.ac.nz/~su/769/vehicles.tar.gz | tar vxz

Every line represents the imports of a particular type of vehicle from a particular country in a particular month. It is just an extended set of the data used in Lab Zero - it includes more HS codes and all original fields as published by Stats NZ.
